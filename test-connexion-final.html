  • AuthContext expose currentUser

  • Résultat : isOnboardingComplete restait toujours false

✅ Corrections Appliquées

  1. Correction de la référence : user → currentUser
  2. Ajout de logs détaillés dans OnboardingContext
  3. Redirection manuelle ajoutée dans Login.js
  4. Gestion explicite du cas "pas d'onboarding sauvegardé"
